Try it out
==========

https://ytimenkov.github.io/instead-block/


Build
=====

.. code-block:: bash

    npm install
    npx ng build

Development server
==================

.. code-block:: bash

    npx ng serve

Then open web browser.

Unit tests
==========

.. code-block:: bash

    npx ng test

End to End tests
================

.. code-block:: bash

    npx ng e2e
